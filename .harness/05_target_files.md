# Target Files

These are the files the AI agent is allowed to create and edit during implementation.

## C# Mod (`mod/StardewAgent/`)

- `mod/StardewAgent/StardewAgent.csproj`
- `mod/StardewAgent/manifest.json`
- `mod/StardewAgent/ModEntry.cs`
- `mod/StardewAgent/GameStateService.cs`
- `mod/StardewAgent/ActionService.cs`
- `mod/StardewAgent/Models/` (any model classes)

## Python Agent (`agent/`)

- `agent/game_client.py`
- `agent/agent.py`
- `agent/actions.py`
- `agent/ai_brain.py`
- `agent/config.py`
- `agent/requirements.txt`
